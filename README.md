# 🧠 Computer Vision Project - Image Processing with Python

## 📌 Description
Ce projet implémente plusieurs techniques de vision par ordinateur pour analyser et améliorer des images en utilisant Python.

## 🚀 Fonctionnalités
- Détection de contours
- Segmentation d’images
- Amélioration de la qualité des images
- Traitement d’images avec OpenCV

## 🛠️ Technologies
- Python
- OpenCV
- NumPy

## 🎯 Objectif
Ce projet a été réalisé dans le cadre de mon apprentissage en vision par ordinateur afin de comprendre les bases du traitement d’images.

## 📈 Améliorations futures
- Détection d’objets
- Intégration de modèles de deep learning
- Interface utilisateur simple
